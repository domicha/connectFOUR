#Connect 4 game implemented in React

### Pre-requirements
- NODE & NPM
- Have gulp installed (version >= 3.9) `npm i gulp -g`

### To get started
1. `cd` into your project directory
2. run `npm install`
3. run `gulp`

### Run tests
In order to run tests, simply run `npm test`
